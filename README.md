# One Home Private Passport

**Prove what matters. Reveal nothing else.**

One Home Private Passport is a privacy-first access application being built with Midnight during the July 17–19, 2026 Midnight Hackathon.

The application allows a user to privately prove that they possess a qualifying credential while revealing only an eligible or not eligible result.

## The Problem

Applications frequently request more wallet and identity information than they actually need.

A person may only need to prove that they possess a qualifying credential, but a traditional wallet connection can expose additional information such as:

- Wallet addresses
- Exact balances
- Complete token holdings
- Collectibles
- Transaction history
- Other wallet activity

One Home Private Passport is designed to minimize that disclosure.

## The Solution

The user privately proves that they possess a qualifying credential.

The application receives only this public result:

- Eligibility verified
- Eligibility not verified

The underlying credential and unnecessary wallet information remain private.

When eligibility is verified, the One Home Founder Room changes from locked to unlocked.

## Hackathon Scope

This repository was created during the July 17–19, 2026 Midnight Hackathon.

All Passport code submitted for the hackathon is being created during the event in this new public repository.

One Home provides the real-world use case, but this repository does not contain the existing One Home application code.

## Planned Demo Flow

1. The Founder Room begins in a locked state.
2. The user selects **Verify Privately**.
3. Midnight verifies the qualifying credential.
4. Only the eligibility result becomes public.
5. The Founder Room unlocks when the result is verified.

## Privacy Goal

The application should not publicly reveal:

- The underlying credential
- Exact wallet balances
- Complete wallet contents
- Transaction history
- Personal identity
- Unrelated digital assets
